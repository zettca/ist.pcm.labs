First Processing Exercise (Bitmapped Image)
Max XP: 400

Using the provided image (PCMLab6.png), do the following (by directly accessing individual pixel and not by using predefined filters or specialized libraries):

1. Create a function that converts the image into grayscale by calculating luminance (consider Y = 0.3R + 0.59G + 0.11B) [100 XP]

2. Create a function that changes the image's contrast according to a given parameter [100 XP]

3. Create a function that segments the image using global thresholding, according to a given parameter [100 XP]

4. Implement other visual effects. Do not forget to be creative! [100 XP]
